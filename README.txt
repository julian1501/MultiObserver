These files contain the BEP of Julian Gootzen: Memory Saving Multi-Observer.

Everything related to the report can be found in the folder: "report".

MATLAB scripts are in the folder: "conventional" and all functions are in the folder: "functions". In order to run the model the file mainClassScript.m should be run, an input dialog will show up where some basic input parameters can be given. In order to change the attack function, change the function: "attackFunction.m".
